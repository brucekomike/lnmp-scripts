# lnmp-scripts
this is a repo for lnmp quick deployment.
## why this repo
- to provide a quick way deploying a bunch of web applications.

## supported platform
- ubuntu

## server minimum requirement
- 2 core
- 2GB RAM

## included softwares
### dependence
- mariadb (database)
- nginx (webserver)
- php (processer)
- certbot (cert provider)

### applications
- mediawiki
- myphpadmin
- phproject
- chyrp-lite
- flarum

## defualt variables
- installation root
